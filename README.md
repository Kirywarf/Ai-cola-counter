# Calorie Counter (WPF, .NET 8)

Полноценное desktop-приложение для Windows для подсчёта калорий и БЖУ на основе локальной базы продуктов и введённого веса.

## Стек
- C# / WPF
- .NET 8 (`net8.0-windows`)
- MVVM
- SQLite (`Microsoft.Data.Sqlite`)
- Интерфейс на русском языке

## Структура проекта
`CalorieCounter/`
- `Models`
- `ViewModels`
- `Views`
- `Services`
- `Data`
- `Helpers`
- `Assets`

## Как открыть проект
1. Откройте `CalorieCounter.sln` в Visual Studio 2022.
2. Проверьте, что установлен .NET 8 SDK и workload для Desktop Development with .NET.
3. Восстановите NuGet-пакеты.

## Как запустить
1. Выберите проект `CalorieCounter` стартовым.
2. Нажмите **F5** (или **Ctrl+F5**).
3. При первом запуске автоматически создаётся SQLite БД в `%LocalAppData%/CalorieCounter/calorie_counter.db`.

## Реализованные функции
- Выбор профиля: создание, редактирование, удаление, вход в профиль.
- Расчёт дневной нормы по формуле Миффлина — Сан Жеора + коэффициенты активности и цель.
- Дашборд: текущая дата, профиль, норма/съедено/остаток калорий, БЖУ, прогресс-бар.
- Добавление еды:
  - из базы продуктов с расчётом по весу;
  - вручную с опцией сохранения продукта в базу.
- База продуктов:
  - стартовый набор 40+ продуктов;
  - поиск, фильтр по категории;
  - добавление, редактирование, удаление.
- Дневник:
  - записи за выбранную дату;
  - редактирование, удаление;
  - копирование записи на другую дату;
  - повторение вчерашнего рациона;
  - итог за день.
- Статистика:
  - калории за 7 и 30 дней;
  - среднее за день;
  - дни с превышением/недобором;
  - самый частый продукт;
  - самый калорийный день.
- Настройки:
  - тема (тёмная/светлая — сохраняется в БД);
  - ручная корректировка дневной нормы калорий;
  - экспорт дневника в CSV;
  - очистка дневника;
  - очистка пользовательских продуктов.

## Как создать профиль
1. Откройте раздел **Профили**.
2. Нажмите **Создать**.
3. Заполните данные (имя, возраст, рост, вес, пол, цель, активность).
4. Нажмите **Рассчитать норму** (или введите вручную).
5. Нажмите **Сохранить**.

## Как добавить продукт в базу
1. Перейдите в **База продуктов**.
2. Нажмите **Новый**.
3. Заполните поля продукта.
4. Нажмите **Сохранить**.

## Как добавить еду в дневник
1. Перейдите в **Добавить еду**.
2. Выберите продукт, укажите вес и приём пищи.
3. Нажмите **Добавить в дневник**.

Либо:
1. Заполните блок ручного добавления.
2. При необходимости отметьте «Сохранить этот продукт в базу».
3. Нажмите **Добавить вручную**.

## Экспорт данных в CSV
1. Откройте **Настройки**.
2. Нажмите **Экспорт в CSV**.
3. Выберите путь сохранения файла.
4. В CSV выгружаются поля: дата, профиль, приём пищи, продукт, вес, калории, белки, жиры, углеводы.
